# OptiMe

RAIN is a full-stack health and wellbeing tracking application developed with React Native Expo, Node.js, Express, MongoDB and MQTT.

The application allows users to track key wellbeing metrics such as mood, stress, anxiety, sleep, screen time, social connection and physical activity. It includes real-time step synchronization from the mobile app using MQTT, backend processing with MongoDB persistence, and a Docker-based development setup for backend services.

## Main features

- React Native Expo mobile and web frontend
- Node.js and Express backend API
- MongoDB data storage
- MQTT broker with Mosquitto
- Real-time pedometer step synchronization
- MQTT processor for backend data handling
- Dockerized backend, MQTT broker and MQTT processor
- Automatic local IP detection for frontend network configuration
- GitHub Actions workflow for frontend testing
